## 🔭 Currently Building

**REPO Tactical Advisor** — an AI-powered tactical assistant for the extraction shooter *R.E.P.O.*

Select parameters (map, enemy type, extractions remaining, money collected) through a structured UI, and the system generates an optimal extraction plan.

**Architecture:** Structured game-state ingestion → context engineering → prompt engineering pipeline → RAG retrieval from curated game knowledge → local LLM inference (Llama 3.1 8B via Ollama) → structured JSON output via FastAPI

**Status:** Working prototype generating responses. Currently improving output quality via prompt engineering, RAG grounding, and evaluation tuning — since an 8B local model with a naive prompt is confident but not always correct. This project is deliberately scoped as an AI engineering exercise (structured I/O, retrieval grounding, context management, iterative evaluation), not an LLM-wrapper demo.
